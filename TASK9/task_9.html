Answers (Short)

1. Why is 12-column grid popular?

Because it is flexible and divisible by many numbers (2, 3, 4, 6).

2. How does spanning help layout?

It allows elements to take more space when needed.

3. Where is this system used?

Websites, dashboards, Bootstrap, admin panels.

4. Can we create layouts without this system?

Yes, it is optional.

5. What happens if column count changes?

Layout structure and spans will change.
